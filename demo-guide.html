▶ Quick Start

1
Open the app — Navigate to delta-voice-app-v1-29f00fe87641.herokuapp.com in Safari (best on iPhone for PWA experience)
2
Add to Home Screen — Tap the Share button → "Add to Home Screen" for the full-screen native app experience
3
Allow microphone access — When prompted, tap "Allow" so the voice assistant can listen

Tap the Assistant button — The microphone icon in the bottom navigation bar activates the voice agent

💡 Demo Tips

Speak Naturally

The agent uses natural language understanding. You don't need to use exact phrases — conversational speech works great.

Wait for the Pulse

The microphone icon pulses when the agent is listening. Start speaking when you see the animated waveform.

Chain Use Cases

Run Use Cases sequentially for the best flow — start with loyalty check, then missing miles, then seat upgrade.

Quiet Environment

For best voice recognition, demo in a quiet room. The agent processes speech-to-text before sending to Agentforce.

Change Demo Persona

Go to More → Personal Info to change the customer name, SkyMiles number, or PNR for different demo scenarios.

Browser Tabs Work Too

The app works in desktop Chrome/Safari as well. The PWA home screen experience is best for mobile demos.

⚡ How It Works

1
Voice Input — User speaks into the app. Browser-native speech recognition converts voice to text in real time.
2
Agentforce Agent — The text is sent to a Salesforce Agentforce agent via the Agent API. The agent has 3 topics (Loyalty, Flight Info, Seat Upgrades) backed by 8 Apex actions.
3
Salesforce Data — Apex actions query live Salesforce objects — Loyalty Management (members, tiers, currencies) and custom Flight/Booking/Seat Map objects.
4
Voice Response — The agent's text response is streamed through Google Gemini TTS for natural voice output. The user hears the response spoken aloud.
